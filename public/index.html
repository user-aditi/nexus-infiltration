📋 How to Run

1

Download or Clone the Project

Choose one of the options above. Either download nexus.js directly or clone the full repository.

# Clone the repository
git clone https://github.com/YOUR_USERNAME/nexus-infiltration.git
cd nexus-infiltration
2

Run the Tool

Execute nexus.js with Node.js. You'll see an interactive menu with 7 options.

# Full mode (recommended)
node nexus.js

# Safe mode (masks sensitive data)
node nexus.js --safe

# Audit only (no file changes)
node nexus.js --audit
3

Select Operation Mode

Press 1 for FULL OPERATION to see everything: system info extraction, CRUD operations, multi-directory scanning, polymorphic infection, and automatic cleanup.
